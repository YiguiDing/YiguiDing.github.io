二阶常系数齐次微分方程求解步骤总结

考虑二阶常系数齐次微分方程：

d2dt2f(t)+bddtf(t)+cf(t)=0 \frac{d^2}{dt^2} f(t) + b \frac{d}{dt} f(t) + c f(t) = 0 dt2d2​f(t)+bdtd​f(t)+cf(t)=0

其中 bbb 和 ccc 是常数。求解步骤如下：

步骤1：假设解的形式

  • 假设解为指数形式： f(t)=Aeλtf(t) = A e^{\lambda t}f(t)=Aeλt （其中 AAA 是常数）。
  • 为什么？ 因为指数函数的导数与自身成比例： ddteλt=λeλt\frac{d}{dt} e^{\lambda t} = \lambda e^{\lambda t}dtd​eλt=λeλt ，高阶导数也类似，这方便代入微分方程后简化为代数方程。

步骤2：代入方程，得到特征方程

  • 将 f(t)=eλtf(t) = e^{\lambda t}f(t)=eλt （不妨忽略常数 AAA ，因为方程是齐次的）代入原方程：

    d2dt2(eλt)+bddt(eλt)+ceλt=0 \frac{d^2}{dt^2} (e^{\lambda t}) + b \frac{d}{dt} (e^{\lambda t}) + c e^{\lambda t} = 0 dt2d2​(eλt)+bdtd​(eλt)+ceλt=0

  • 计算导数：

    ddt(eλt)=λeλt,d2dt2(eλt)=λ2eλt \frac{d}{dt} (e^{\lambda t}) = \lambda e^{\lambda t}, \quad \frac{d^2}{dt^2} (e^{\lambda t}) = \lambda^2 e^{\lambda t} dtd​(eλt)=λeλt,dt2d2​(eλt)=λ2eλt

  • 代入并整理：

    λ2eλt+bλeλt+ceλt=0 \lambda^2 e^{\lambda t} + b \lambda e^{\lambda t} + c e^{\lambda t} = 0 λ2eλt+bλeλt+ceλt=0

  • 除以非零因子 eλte^{\lambda t}eλt （因为 eλt≠0e^{\lambda t} \neq 0eλt=0 ），得到特征方程：

    λ2+bλ+c=0 \lambda^2 + b\lambda + c = 0 λ2+bλ+c=0

步骤3：求解特征根

  • 特征方程为二次方程：

    λ=−b±b2−4ac2 \lambda = \frac{-b \pm \sqrt{b^2 - 4ac}}{2} λ=2−b±b2−4ac​​

  • 根据判别式 D=b2−4ac=b2−4cD = b^2 - 4ac= b^2 - 4cD=b2−4ac=b2−4c 的值，特征根分为三种情况：
    1. 两个不同的实根：当 D>0D > 0D>0 ，根为 λ1\lambda_1λ1​ 和 λ2\lambda_2λ2​ （实数和不等）。
    2. 重根：当 D=0D = 0D=0 ，根为 λ1=λ2=λ\lambda_1 = \lambda_2 = \lambdaλ1​=λ2​=λ （实数和相等）。
    3. 一对共轭复根：当 D<0D < 0D<0 ，根为 α±βi\alpha \pm \beta iα±βi ，其中实部 α=−b2\alpha = -\frac{b}{2}α=−2b​ ，虚部 β=∣D∣2=∣b2−4ac∣2\beta = \frac{\sqrt{|D|}}{2} = \frac{ \sqrt{|b^2 - 4ac|} }{2}β=2∣D∣​​=2∣b2−4ac∣​​ 。

步骤4：根据特征根类型写出通解

通解的形式取决于特征根的类型：

情况1：两个不同的实根（ λ1≠λ2\lambda_1 \neq \lambda_2λ1​=λ2​ ）

  • 为什么？
    因为微分方程是线性齐次的，叠加原理适用：两个独立的特解（eλ1te^{\lambda_1 t}eλ1​t 和 eλ2te^{\lambda_2 t}eλ2​t） 的线性组合构成通解。常数 AAA 和 BBB 由初始条件确定。

情况2：重根（ λ1=λ2=λ\lambda_1 = \lambda_2 = \lambdaλ1​=λ2​=λ ）

  • 通解： f(t)=(A+Bt)eλtf(t) = (A + B t) e^{\lambda t}f(t)=(A+Bt)eλt
  • 为什么？
    当根重时，只有一个独立的特解 eλte^{\lambda t}eλt 。第二个独立的特解通过引入线性因子 ttt 获得：
  • 如何找到第二个解 teλtt e^{\lambda t}teλt ？

    • 设第二个解为 f2(t)=v(t)eλtf_2(t) = v(t) e^{\lambda t}f2​(t)=v(t)eλt （其中 v(t)v(t)v(t) 是待定函数）。

    f(t)=Pe(α+iβ)t+Qe(α−iβ)t f(t) = Pe^{(\alpha + i\beta)t}+Qe^{(\alpha - i\beta)t} f(t)=Pe(α+iβ)t+Qe(α−iβ)t

解的物理意义解释（以机械振动或电路系统为例）

二阶常系数齐次微分方程常见于物理系统，如弹簧-质量-阻尼系统或RLC电路（电压或电流的方程）。方程的项分别对应：

  • d2dt2f(t)\frac{d^2}{dt^2} f(t)dt2d2​f(t) ： 惯性或电感效应（加速度相关）。
  • bddtf(t)b \frac{d}{dt} f(t)bdtd​f(t) ： 阻尼或电阻效应（速度相关）。
  • cf(t)c f(t)cf(t) ： 弹性或电容效应（位置相关）。

根的类型对应系统的不同阻尼状态：

  • 两个不同的实根（过阻尼，例如 RLC 过阻尼或高阻尼机械系统）：

    • 通解形式： f(t)=Aeλ1t+Beλ2tf(t) = A e^{\lambda_1 t} + B e^{\lambda_2 t}f(t)=Aeλ1​t+Beλ2​t ，其中 λ1\lambda_1λ1​ 和 λ2\lambda_2λ2​ 均为负实数（衰减）。
    • 为什么是线性组合？ 物理上，系统有两个独立的指数衰减模式（例如，电容和电感的能量以不同速率释放）。通解是这两个模式的叠加，因为系统的行为可分解为两个一阶衰减（如 "RC 衰减" 和 "RL 衰减" 的耦合）。每个指数项对应一个衰减通道。
    • 物理意义： 系统平滑地返回平衡点，无振荡。例如：
      • RLC 过阻尼响应：电流或电压缓慢衰减（无振荡）。
      • 机械系统：重物缓慢返回静止位置（如门闭门器过阻尼时）。
    • 数学原因： 特征根不同，解空间维数为 2，两个特解 eλ1te^{\lambda_1 t}eλ1​t 和 eλ2te^{\lambda_2 t}eλ2​t 线性无关，构成基。

  • 重根（临界阻尼，例如 RLC 临界阻尼或理想阻尼机械系统）：

    • 通解形式： f(t)=(A+Bt)eλtf(t) = (A + B t) e^{\lambda t}f(t)=(A+Bt)eλt 。
    • 为什么有线性因子 ttt ？ 在临界阻尼时，两个根重合，衰减速率相同，系统无法分离为两个独立模式。线性因子 ttt 引入"记忆"效应，表示系统在平衡点附近有轻微"过冲"但无振荡。
    • 物理意义： 系统以最快速度返回平衡点而不振荡。例如：
      • RLC 临界阻尼响应：电流或电压快速衰减到零（无振荡）。
      • 机械系统：质量以最小时间返回静止位置（如汽车悬架临界调校）。
    • 数学原因： 降阶法显示，当特征根重时，方程简化为 v′′=0v'' = 0v′′=0 ，给出线性 v(t)v(t)v(t) 。

  • 共轭复根（欠阻尼，例如 RLC 欠阻尼或低阻尼机械系统）：

    • 通解形式： f(t)=eαt(Acos⁡(βt)+Bsin⁡(βt))f(t) = e^{\alpha t} \left( A \cos(\beta t) + B \sin(\beta t) \right)f(t)=eαt(Acos(βt)+Bsin(βt)) ，其中 α=−b2\alpha = -\frac{b}{2}α=−2b​ （通常负，表示衰减）。
    • 物理意义： 系统振荡衰减（阻尼振荡）。实部 α\alphaα 控制振幅衰减率，虚部 β\betaβ 控制振荡频率。例如：
      • RLC 欠阻尼响应：电流或电压振荡衰减（如无线电调谐电路）。
      • 机械系统：质量围绕平衡点振动并逐渐停止（如弹簧振子带小阻尼）。
    • 零阻尼特殊情况：若 b=0b = 0b=0 ，则 α=0\alpha = 0α=0 ，解为 Acos⁡(βt)+Bsin⁡(βt)A \cos(\beta t) + B \sin(\beta t)Acos(βt)+Bsin(βt) ，表示持续简谐振荡（如理想LC电路或无阻尼摆）。

为什么通解是特解的线性组合？

  • 核心数学原因： 线性齐次微分方程的解空间是向量空间（齐次原理）。特征方程提供独立特解：
    • 实根不同：两个指数特解线性无关。
    • 重根：一个指数特解加一个衍生解（通过降阶法）。
    • 复根：通过三角函数形式确保实值解。
  • 解的线性组合覆盖所有可能初始条件（如 f(0)f(0)f(0) 和 f′(0)f'(0)f′(0) )。

此方法通用，适用于任何二阶常系数齐次线性微分方程。初始条件用于确定常数 AAA 和 BBB ，从而得到特解。

降阶法（Reduction of Order）推导过程详解

降阶法是求解重根情况下第二个线性无关特解的核心方法。以下是完整推导（针对二阶常系数齐次微分方程的重根情形）：

问题设定

考虑方程：

d2f(t)dt2+bdf(t)dt+cf(t)=0 \frac{d^2 f(t)}{dt^2} + b \frac{d f(t)}{dt} + c f(t) = 0 dt2d2f(t)​+bdtdf(t)​+cf(t)=0

特征方程 λ2+bλ+c=0\lambda^2 + b\lambda + c = 0λ2+bλ+c=0 有重根 λ1=λ2=λ\lambda_1 = \lambda_2 = \lambdaλ1​=λ2​=λ，其中：

λ=−b2,且b2−4ab=0. \lambda = -\frac{b}{2}, \quad \text{且} \quad b^2 - 4ab = 0. λ=−2b​,且b2−4ab=0.

已知一个特解 f1(t)=eλtf_1(t) = e^{\lambda t}f1​(t)=eλt，目标是找到第二个线性无关的特解 f2(t)f_2(t)f2​(t).

    v′′+0⋅v′+0⋅v=0⇒v′′(t)=0 v'' + 0 \cdot v' + 0 \cdot v = 0 \quad \Rightarrow \quad \boxed{v''(t) = 0} v′′+0⋅v′+0⋅v=0⇒v′′(t)=0​

    • 对 v′′(t)=0v''(t) = 0v′′(t)=0 积分两次：

    v′(t)=C1,v(t)=C1t+C2 v'(t) = C_1, \quad v(t) = C_1 t + C_2 v′(t)=C1​,v(t)=C1​t+C2​

    其中 C1C_1C1​, C2C_2C2​ 为常数。
    • 第二个特解为：

    f2(t)=v(t)eλt=(C1t+C2)eλt f_2(t) = v(t) e^{\lambda t} = (C_1 t + C_2) e^{\lambda t} f2​(t)=v(t)eλt=(C1​t+C2​)eλt

    • 线性无关性验证：

    ◦ 当 C1=0C_1 = 0C1​=0, C2=1C_2 = 1C2​=1 时，得 f1(t)=eλtf_1(t) = e^{\lambda t}f1​(t)=eλt（第一个解）。

    ◦ 当 C1=1C_1 = 1C1​=1, C2=0C_2 = 0C2​=0 时，得 f2(t)=teλtf_2(t) = t e^{\lambda t}f2​(t)=teλt（第二个解）。

    两者线性无关（因 teλteλt=t\frac{t e^{\lambda t}}{e^{\lambda t}} = teλtteλt​=t 非常数）。

  6. 通解

    f(t)=A⋅f1(t)+B⋅f2(t)=(A+Bt)eλt f(t) = A \cdot f_1(t) + B \cdot f_2(t) = \boxed{(A + B t) e^{\lambda t}} f(t)=A⋅f1​(t)+B⋅f2​(t)=(A+Bt)eλt​

    其中 A=C2A = C_2A=C2​, B=C1B = C_1B=C1​ 为任意常数。

为何降阶法有效？

• 数学本质：通过引入函数 v(t)v(t)v(t)，将问题从求解二阶微分方程降阶为求解 v′′(t)=0v''(t) = 0v′′(t)=0（一阶可解问题）。

• 物理对应：在重根情况下（如临界阻尼），系统响应需要额外项 teλtt e^{\lambda t}teλt 描述"临界过冲"行为（如最短时间返回平衡点但无振荡）。

对比其他方法

• 参数变易法（Variation of Parameters）：用于非齐次方程，通过变易齐次解的常数求特解，不适用于此处的齐次方程求通解。

• 特征方程法推广：直接由特征根形式导出解的结构（复根时需欧拉公式转换），但无法解释重根下 teλtt e^{\lambda t}teλt 的来源。

总结：降阶法是求解重根情形下线性无关特解的标准方法，依赖于特征根的性质和微分方程的线性结构。

解的物理意义（以振动或电路为例）

  • 齐次解 fh(t)f_h(t)fh​(t): 系统的 自由响应（固有行为），

    • 过阻尼（实根）：能量单调衰减（如门缓慢关闭），
    • 临界阻尼（重根）：最快无振荡返回平衡（如汽车悬架临界调校），
    • 欠阻尼（复根）：振荡衰减（如弹簧振子带摩擦）。

  • 特解 fp(t)f_p(t)fp​(t): 系统的 强迫响应（由输入 g(t)g(t)g(t) 驱动），

    • 体现系统对外部激励的稳态输出（如电路对交流电源的响应），
    • 形式取决于 g(t)g(t)g(t)（如常数输入→常数稳态；正弦输入→同频正弦振荡）。

  • 通解 f(t)f(t)f(t):

    总响应=自由响应（瞬态）+强迫响应（稳态） \text{总响应} = \text{自由响应（瞬态）} + \text{强迫响应（稳态）} 总响应=自由响应（瞬态）+强迫响应（稳态）

    • 特解：电源 g(t)g(t)g(t) 驱动的稳态电流/电压。
