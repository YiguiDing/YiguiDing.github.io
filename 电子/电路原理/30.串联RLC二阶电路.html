串联RLC二阶电路

零状态RLC串联电路的自然响应

  • 拓扑约束
    • KVL: uC=uL+uR
    • KCL: iC=iL=iR=i
  • 元件约束
    • iC=CduCdt
    • uL=LdiLdt
    • R=uRiR

列写方程：

uL+uR=uCLddtiL+RiR=uCLddt(iC)+R(iC)=uCLddt(CduCdt)+R(CduCdt)=uCLCd2uCd2t+RCduCdt=uCLCd2uCd2t+RCduCdt−uC=0d2uCd2t+RLduCdt−1LCuC=0

定义：2α=RL，ω02=1LC，得：

d2uCd2t+2αduCdt−ω02uC=0

特征方程为

λ2+2αλ−ω02=0

判别式为

Δ=b2−4ac=(2α)2−4×1×(ω02)=4α2−4ω02=4(α2−ω02)

特征根为

λ=−b±Δ2a=−(2α)±4(α2−ω02)2×1=−2α±2α2−ω022×1=−α±α2−ω02

根据判别式的情况，微分方程的解为：

过阻尼情况： 当α2>ω02 ( 即 Δ=4(α2−ω02)>0 ) 时

λ1,λ2=−α±α2−ω02uC=Aeλ1t+Beλ2t

临界阻尼情况： 当α2=ω02 ( 即 Δ=4(α2−ω02)=0 ) 时

λ=−α±α2−ω02=−α±0=−αuC=Aeλt+Bteλt

欠阻尼情况： 当α2<ω02 ( 即 Δ=4(α2−ω02)<0 ) 时

λ1,λ2=−α±α2−ω02=−α±(−1)×(ω02−α2)=−α±−1×(ω02−α2)=−α±i×(ω02−α2)ωd=(ω02−α2)uC=e−αt(Acos⁡(ωdt)+Bsin⁡(ωdt))R=A2+B2ϕ=tan−1⁡ABuC=Re−αtsin⁡(ωdt+ϕ)

零阻尼情况： 当α=0 ( 即 Δ=4(α2−ω02)=−4ω02<0 ) 时

λ1,λ2=−α±α2−ω02=−0±02−ω02=±−1×ω02=±−1×ω02=±iω02(特征根为纯虚数)ωd=ω02=ω0uC=e−αt(Acos⁡(ωdt)+Bsin⁡(ωdt))=e−0t(Acos⁡(ωdt)+Bsin⁡(ωdt))=Acos⁡(ωdt)+Bsin⁡(ωdt)=Acos⁡(ω0t)+Bsin⁡(ω0t)R=A2+B2θ=tan−1⁡ABuC=Re−αtsin⁡(ωdt+θ)=Re−0tsin⁡(ωdt+θ)=Rsin⁡(ωdt+θ)=Rsin⁡(ω0t+θ)

根据拓扑约束和元件约束写出微分方程；根据特征方程的解写出微分方程的通解形式。
