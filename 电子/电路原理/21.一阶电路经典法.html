一阶电路经典法

RC一阶电路

案例

1.列写方程（拓扑约束+元件约束）

  • uR+uc=Usu_R + u_c = U_suR​+uc​=Us​
  • uR=iRu_R = iRuR​=iR
  • i=Cd(uc)dti = C\frac{d(u_c)}{dt}i=Cdtd(uc​)​

RCd(uc)dt+uc=Us RC\frac{d(u_c)}{dt} + u_c = U_s RCdtd(uc​)​+uc​=Us​

2.求解方程

齐次项的解

  • 假设解的形式为uc=Aeλtu_c = Ae^{\lambda t}uc​=Aeλt
  • 带入原方程得到特征方程: RCλ+1=0RC\lambda + 1 = 0RCλ+1=0
  • 特征根: λ=−1RC\lambda = -\frac{1}{RC}λ=−RC1​
  • 微分方程的齐次通解：

uc′=Aeλt=Ae−tRC u_c' = Ae^{\lambda t} = Ae^{-\frac{t}{RC}} uc′​=Aeλt=Ae−RCt​

非齐次项的解

  • UsU_sUs​是一个常数
  • 显然可得微分方程的非齐次特解：

uc′′=Us u_c''=U_s uc′′​=Us​

通解=齐次通解+非齐次特解

uc=uc′′+uc′=Us+Ae−tRC \begin{align*} u_c&=u_c''+u_c' \\ &=U_s + Ae^{-\frac{t}{RC}} \end{align*} uc​​=uc′′​+uc′​=Us​+Ae−RCt​​

3.确定初值

  • 换路前：uc(0−)=U0u_c(0^-)=U_0uc​(0−)=U0​
  • 换路后：uc(0+)=uc(0−)=U0u_c(0^+)=u_c(0^-)=U_0uc​(0+)=uc​(0−)=U0​

4.确定系数

将 uc(0+)=U0u_c(0^+)=U_0uc​(0+)=U0​ 带入方程确定系数A:
uc=Us+Ae−tRCu_c = U_s + Ae^{-\frac{t}{RC}}uc​=Us​+Ae−RCt​

t=0时：
U0=Us+Ae−0RCU_0 = U_s + Ae^{-\frac{0}{RC}}U0​=Us​+Ae−RC0​
U0=Us+AU_0 = U_s + AU0​=Us​+A

得：

A=U0−Us A=U_0-U_s A=U0​−Us​

5.代入系数

uc=Us+(U0−Us)e−tRC u_c = U_s + (U_0-U_s)e^{-\frac{t}{RC}} uc​=Us​+(U0​−Us​)e−RCt​

RC一阶电路的时间常数

  • 对RC一阶电路的两个支路量求解可以发现，
  • 他们的指数项时相同的，意味着其变化速率也是相同的。
  • 所以可以定义一个变量τ=RCτ=RCτ=RC 称RC一阶电路的时间常数，单位秒。

uc=Us+(U0−Us)e−tRC u_c = U_s + (U_0-U_s)e^{-\frac{t}{RC}} uc​=Us​+(U0​−Us​)e−RCt​

uc=Us+(U0−Us)e−tτ u_c = U_s + (U_0-U_s)e^{-\frac{t}{τ}} uc​=Us​+(U0​−Us​)e−τt​

RL一阶电路

案例

1.列写方程（拓扑约束+原件约束）

KVL

  • −UR=uL-U_R = u_L−UR​=uL​
  • UR=R∗iU_R = R*iUR​=R∗i

元件约束

  • uL=Ldidtu_L = L\frac{di}{dt}uL​=Ldtdi​

整理

−Ri=LdidtLdidt−Ri=0LRdidt−i=0 \begin{align*} - Ri &= L\frac{di}{dt} \\ L\frac{di}{dt} - Ri &= 0 \\ \frac{L}{R}\frac{di}{dt} - i &= 0 \\ \end{align*} −RiLdtdi​−RiRL​dtdi​−i​=Ldtdi​=0=0​

2.求解方程

齐次项的解

  • 假设解的形式为iL=Aeλti_L = Ae^{\lambda t}iL​=Aeλt
  • 带入原方程得到特征方程: LRλ+1=0\frac{L}{R}\lambda + 1 = 0RL​λ+1=0
  • 特征根: λ=−1LR=−RL\lambda = -\frac{1}{\frac{L}{R}}=-\frac{R}{L}λ=−RL​1​=−LR​
  • 齐次通解：iL=Aeλt=Ae−RLti_L = Ae^{\lambda t} = Ae^{-\frac{R}{L}t}iL​=Aeλt=Ae−LR​t

3.确定初值

在t=0时刻，

i(0)=i(0+)=i(0−)=UsR1+Ri(0)=i(0^+)=i(0^-)=\frac{U_s}{R_1+R}i(0)=i(0+)=i(0−)=R1​+RUs​​

4.确定系数

iL=Ae−RLtA=UsR1+R \begin{align*} i_L &= Ae^{-\frac{R}{L}t} \\ A &= \frac{U_s}{R_1+R} \end{align*} iL​A​=Ae−LR​t=R1​+RUs​​​

5.代入系数

iL=UsR1+Re−RLt i_L = \frac{U_s}{R_1+R}e^{-\frac{R}{L}t} iL​=R1​+RUs​​e−LR​t

RL一阶电路的时间常数

iL=UsR1+Re−RLt i_L = \frac{U_s}{R_1+R}e^{-\frac{R}{L}t} iL​=R1​+RUs​​e−LR​t

定义τ=LRτ=\frac{L}{R}τ=RL​ 称RL一阶电路的时间常数，单位秒。

iL=UsR1+Re−tτ i_L = \frac{U_s}{R_1+R}e^{-\frac{t}{τ}} iL​=R1​+RUs​​e−τt​

关于τ的讨论

工程观点

  • t=3τ，e−tτe^{-\frac{t}{τ}}e−τt​ =0.05
  • t=5τ，e−tτe^{-\frac{t}{τ}}e−τt​ =0.007

  • 所以可以认为，经过3~5τ的时间后，电路的过度过程完全结束。

规律

  • τ越小，电路的过渡过程越快
  • τ越大，电路的过渡过程越慢

区别

  • τ=RCτ=RCτ=RC R越大，τ越大，R增大的作用是减缓电容上电荷被中和的速度
  • τ=LRτ=\frac{L}{R}τ=RL​ R越小，τ越大，R减小的作用是增大L的续流能力保持电感建立的磁场。

总结

一阶电路经典法求解

  1. 列写所关心支路量x(t)x(t)x(t)的常微分方程（拓扑约束+原件约束）
  2. 根据换路定律，求x(0+)x(0^+)x(0+)时刻的数值
  3. 求微分方程
    • 方程是齐次微分方程，求其齐次通解
    • 方程是非齐次微分方程，求其齐次通解+非齐次特解
  4. 通过x(0+)x(0^+)x(0+)来确定齐次通解的待定系数
