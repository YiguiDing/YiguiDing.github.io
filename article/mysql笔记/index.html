mySql笔记

mySql8.0

  • 列出所有数据库:show databases;
  • 清空MySQL——root密码
    1. 停止 MySQL 任务 net stop MySQL
    2. mysqld 命令 mysqld –console –skip-grant-tables –shared-memory
    3. 无密码进入mysql mysql -u root
    4. 清空root 密码 UPDATE user SET authentication_string=”” WHERE user=’root’;
    5. 重新载入授权表。FLUSH PRIVILEGES;
      • 如果你不使用该命令，你就无法使用新创建的用户来连接mysql服务器，除非你重启mysql服务器。
  • 修改用户密码
    1. use mysql;
    2. alter user root@localhost identified by ‘密码’;
    3. FLUSH PRIVILEGES;
