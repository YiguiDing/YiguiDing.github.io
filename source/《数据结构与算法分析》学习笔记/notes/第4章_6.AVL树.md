# 第4章_6.AVL树



## AVL树 
* AVL(Adelson-Velskii and Landis)树,名字取自其发明者G.M.Adelson-Velsky 和 E.M. Landis的首字母
* 是一种带有**平衡条件**的二叉查找树
* 这个平衡条件必须要容易保持，且必须保证树的深度是O(log N)
* 在每个节点中记录高度信息
* 通常定义空子树的高度为-1
  + 高度为0的AVL树，意味着应该有一个节点，0个子树
  + 高度为1的AVL树，意味着应该有一片或两片叶子。
  + 空树的高度为-1的约定实际上在AVL树中有一些实际用途,因为它简化了平衡因子的计算以及何时旋转子代。
* 每个节点的左子树和右子树高度最多相差1
  + 不符合该要求的不是AVL树![](/《数据结构与算法分析》学习笔记/attachments/Clipboard_2022-05-31-11-18-54.png)
* 一些不好的平衡条件：
  + 要求根节点的左右子树具有相同的高度
    + 不强求深度要浅
    + 最坏情况：![](/《数据结构与算法分析》学习笔记/attachments/Clipboard_2022-05-29-07-06-48.png)
    + 根据定义允许的情况：![](/《数据结构与算法分析》学习笔记/attachments/Clipboard_2022-05-31-09-04-11.png)
  + 要求每个节点的左右子树的高度必须相同
    + 能够保证树的深度，但条件过于严格
## **AVL树高度h和节点数S(h)之间的关系**
+ 高度为-1的树,节点数至少为0 至多为0
  + **Min(S(-1))=0**
  + **Max(S(-1))=0**
  + 左图为反映了至少有多少个节点，
  + 右图反映了至多有多少个节点，
  ![](/《数据结构与算法分析》学习笔记/attachments/Clipboard_2022-05-31-12-19-50.png)
+ 高度为 0的树,节点数至少为1 至多为1
  + **Min(S(0))=1**
  + **Max(S(0))=1**
  + 左图为反映了至少有多少个节点，
  + 右图反映了至多有多少个节点，
  ![](/《数据结构与算法分析》学习笔记/attachments/Clipboard_2022-05-31-12-19-26.png)
+ 高度为 1的树,节点数至少为2 至多为3
  + Min(S(**1**))=Min(S(**0**))+Min(S(**-1**))+1=2
  + Max(S(**1**))=Max(S(**0**))+Max(S(**0**))+1=3
  + 左图为反映了至少有多少个节点，其中树的左子树就是n-1的情况 右子树是n-2的情况
  + 右图反映了至多有多少个节点，其中树的左子树和右子树都是n-1的情况
  ![](/《数据结构与算法分析》学习笔记/attachments/Clipboard_2022-05-31-11-48-10.png)
+ 高度为 2的树,节点数至少为4 至多为7
  + Min(S(**2**))=Min(S(**1**))+Min(S(**0**))+1=4
  + Max(S(**2**))=Max(S(**1**))+Max(S(**1**))+1=7
  + 左图为反映了至少有多少个节点，其中树的左子树就是n-1的情况 右子树是n-2的情况
  + 右图反映了至多有多少个节点，其中树的左子树和右子树都是n-1的情况
  ![](/《数据结构与算法分析》学习笔记/attachments/Clipboard_2022-05-31-12-12-54.png)
+ 高度为 3的树,节点数至少为7 至多为5
  + Min(S(**3**))=Min(S(**2**))+Min(S(**1**))+1=7
  + Max(S(**3**))=Max(S(**2**))+Max(S(**2**))+1=5
  + 左图为反映了至少有多少个节点，其中树的左子树就是n-1的情况 右子树是n-2的情况
  + 右图反映了至多有多少个节点，其中树的左子树和右子树都是n-1的情况
  ![](/《数据结构与算法分析》学习笔记/attachments/Clipboard_2022-05-31-12-11-51.png)
+ 高度为 4的树,节点数至少为4+7+1=12 至多为5+5+1=11
  + Min(S(**4**))=Min(S(**3**))+Min(S(**2**))+1=12
  + Max(S(**4**))=Max(S(**3**))+Max(S(**3**))+1=11
  + 左图为反映了至少有多少个节点，其中树的左子树就是n-1的情况 右子树是n-2的情况
  + 右图反映了至多有多少个节点，其中树的左子树和右子树都是n-1的情况
  ![](/《数据结构与算法分析》学习笔记/attachments/Clipboard_2022-05-31-12-06-39.png)
+ 所以可以总结出其关系类似于斐波那契数列，
  + 定义S(-1)=0;S(0)=1;高度为h的AVL树的节点数至少为：**S(h)=S(h-1)+S(h-2)+1**
  + 定义S(-1)=0;S(0)=1;高度为h的AVL树的节点数至多为：**S(h)=S(h-1)+S(h-1)+1**
+ 若问：高度为9的AVL树至少有多少个节点、至多有多少个？
  若定义S(-1)=0;S(0)=1;
  则至少有min(S(9))=min(S(9-1))+min(S(9-2))+1个节点
  至多有max(S(9))=max(S(9-1))+max(S(9-1))+1个节点
  ![](/《数据结构与算法分析》学习笔记/attachments/Clipboard_2022-05-31-13-22-18.png)
